Linkajou

Basic Link GH plugin and Rhino plugin

Documentation using DocFX:

Install DocFx

   docfx --build
   docfx --serve

Features:

Giraffe import/export (GeoJSON) Facade generation ala BuildingSketcher style GraphTree setup for relations between Plot->Building->Apartment->Room->Windows

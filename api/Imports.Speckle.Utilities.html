DataTreeToSpeckle(GH_Structure<IGH_Goo>, ISpeckleConverter, Action, Int32)

Converts a Grasshopper Grasshopper.DataTree<T> to a Speckle.Core.Models.Base object. All paths in the will become the name of a property in the Speckle.Core.Models.Base object, with their value set as a List<T> containing the items of that path.

Declaration
public static Base DataTreeToSpeckle(GH_Structure<IGH_Goo> dataInput, ISpeckleConverter converter, Action onConversionProgress = null, int chunkLength = 1000)
Parameters
Type Name Description
Grasshopper.Kernel.Data.GH_Structure<Grasshopper.Kernel.Types.IGH_Goo> dataInput

The input Grasshopper.Kernel.Data.GH_Structure<T>
Speckle.Core.Kits.ISpeckleConverter converter

The converter to use for each path's items.
Action onConversionProgress

Optional action to report progress to the caller.
Int32 chunkLength

The length of the chunks to be created for each path list.
Returns
Type Description
Speckle.Core.Models.Base

A new Base object where all dynamic properties have a list as value, and their names match the pattern described in the example.
Remarks

Since the amount of items in each Grasshopper.DataTree<T> path can greatly vary, properties will be automatically detached. This is done by adding a prefix to every path property following the pattern @(DETACH_COUNT).

Examples

For the path {0;0;0} the resulting property name will be @(1000){0;0;0}.
